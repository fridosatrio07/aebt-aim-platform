# PR Review Checklist

- [ ] Scope matches the task packet.
- [ ] No out-of-scope feature was added.
- [ ] No protected file changed without approval.
- [ ] Build/lint/typecheck/test checks passed or exceptions are documented.
- [ ] Tenant isolation checked.
- [ ] RBAC checked.
- [ ] Audit trail checked.
- [ ] Migration check passed where applicable.
- [ ] UI empty/loading/error/access denied states checked where applicable.
- [ ] No invented compliance/RBI/legal/certification logic introduced.
- [ ] No unsupported final technical/safety decision introduced.
- [ ] Documentation updated where applicable.
- [ ] Task checklist updated only if complete.
- [ ] Handoff summary completed.
